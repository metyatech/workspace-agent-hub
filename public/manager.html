最初の使い方

ここでは、思いついたことをそのまままとめて送って大丈夫です

何の task かを先に決める必要はありません。AI が既存の task に振り分けるか、新しい task を作るか、確認が必要かを整理します。 整理した task は、そのまま built-in Manager が実際の作業まで進めます。

1. 下の送信ボタンを開いて、やりたいことをそのまま書く
2. AI が task ごとに分けて、必要なものを上に並べる
3. 返事が必要なものから順に確認する
