Workspace Agent Hub

スマホでも、同じ AI セッションをそのまま続ける

ここでは最初に新しい session を始めるか、いま動いている session を選びます。選んだ session の出力は下に見え続け、 追加の指示や停止もその場でできます。

1. 上で session を始めるか一覧から選ぶ
2. 出力を見ながら一言ずつ追加指示する
3. PC でもスマホでも同じ session を続きから開ける
新しい session を始める

まず 1 本立ち上げたいときはここからです。タイトルは空でもよく、 そのまま開始フォームへ移動できます。

開始フォームへ移動して、そのまま入力できます。
続きから選ぶ

前に使っていた session の続きを送りたいときは、一覧から選びます。 検索欄まで移動して、すぐ絞り込めます。

session 検索欄へ移動して、続きに戻ります。
Manager を開く

Hub は目の前の 1 本の session を続ける画面です。Manager は複数の依頼やスレッドを整理するときの画面です。PC でもスマホでも同じ Manager を開けます。

クリックするとそのまま Manager へ移動します。
接続状態を確認中

最初の同期が終わると、ここにオンライン状態と再接続の目安が出ます。

スマホに追加

HTTPS で開いているときは、ホーム画面アプリとして追加できます。PC ではデスクトップかスタートメニューの Workspace Agent Hub を使うと、毎回コマンドを打たずに開けます。

まだインストール待ちです。
この端末の扱い

通知を許可すると、画面を閉じていても選択中 session の更新に気づけます。

通知は未設定です。
ロックすると、このブラウザに保存した接続情報とキャッシュを消します。
スマホから開く

同じ Tailscale 内のスマホでは、この URL をそのまま開けます。QR は最初の共有が必要なときだけ使います。

必要なときだけ QR を表示します。

URL をホーム画面に追加しておくと、次からはそのまま開けます。

必要なときだけ、このコマンドで Tailscale Serve を有効にできます。
